# FarmNive

AI-Powered Modular Hydroponic Farming System for Sustainable Agriculture

## About

FarmNive (formerly AgroVision) is an award-winning agricultural technology startup that won first place and $100,000 in seed funding at the 2025 Capital One/Delaware State University Innovation Venture Competition.

Our AI-powered modular hydroponic farming system addresses critical challenges in modern agriculture:
- Food insecurity in urban and rural communities
- Climate volatility and extreme weather impacts
- Agricultural resource scarcity and sustainability

## Technology Features

- **Modular Hydroponics**: Scalable infrastructure with automated water and nutrient management
- **AI-Powered Detection**: Computer vision and machine learning for disease detection and crop health monitoring
- **Digital Dashboard**: User-friendly interface with mobile integration
- **AR/VR Training**: Immersive training and monitoring tools
- **Resource Optimization**: Smart water and nutrient management
- **Data Analytics**: Comprehensive insights for optimizing growing conditions

## Deployment

This site is built for GitHub Pages deployment.

### Prerequisites
- GitHub account
- Git installed on your local machine

### Setup Instructions

1. **Initialize Git Repository** (if not already done)
   ```bash
   git init
   git add .
   git commit -m "Initial commit: FarmNive website"
   ```

2. **Create GitHub Repository**
   - Go to [GitHub](https://github.com) and create a new repository named `farmnive`
   - Do not initialize with README, .gitignore, or license

3. **Push to GitHub**
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/farmnive.git
   git branch -M main
   git push -u origin main
   ```

4. **Enable GitHub Pages**
   - Go to your repository settings
   - Navigate to "Pages" section
   - Under "Source", select "main" branch
   - Select "/ (root)" as the folder
   - Click "Save"

5. **Access Your Site**
   - Your site will be available at: `https://YOUR_USERNAME.github.io/farmnive/`
   - It may take a few minutes for the site to be published

### Local Development

To view the site locally, simply open `index.html` in your web browser:

```bash
open index.html
```

Or use a local server:

```bash
python -m http.server 8000
# Then visit http://localhost:8000
```
